@package LIZY 1.0.0

A little PIM really fast in I/O treatment. Just one channel for the moment, but more will come soon.

INSTALLATION

Install nodejs https://nodejs.org/en/download/package-manager/
Install mongoDb https://docs.mongodb.com/manual/installation/

Download LIZY sources from https://github.com/Raphael-johanne/lizy

Run a nodejs server with this command from the LIZY main directory installation

nodejs app.js

Execute this command line to install LIZY PIM

nodejs console.js connector:install

That's it, enjoy :)

PS : "TypeError: Cannot read property 'Kerberos' of undefined at Object.<anony..." is not important
